# Betting Company Report – BI Report
A Power BI report analysing sportsbook performance, with a focus on GGR, profitability, and bet-type segmentation (singles vs. multiples).

## Repository structure
```
├── data/
│   └── bets.csv              # Source dataset (betting records)
├── Betting company report.pdf   # Client-facing report (print-ready PDF)
└── Betting company report.pbit  # Power BI template file
```

## Files
#### data/bets.csv
Raw betting records used as the data source for the report.
#### SportsBettingReport.pbit
Power BI template file. Open this in Power BI Desktop and point it to data/bets.csv when prompted to load data. All measures, visuals, and report pages are included.
#### SportsBettingReport.pdf
Export of the final report, formatted for print and client delivery.
